MEDIA QUERY

- Las media queries son estilos que se aplican si se cumplen ciertas reglas. Las sintaxis es como sigue:
@media (max-width: 767px) { p { color: blues; } }
Lo que significa que si se cumple que max-width es 767, entonces se ponen los párrafos de color blue. Uno muy usado es @media (orientation: portrait) {...}.
- Las reglas se pueden combinar con operadores lógicos (and, or se pone con ",")
- Se debe procurar no poner diferentes reglas que se cumplan a la vez en diferentes estilos, de forma que se apliquen dos estilos a la vez, esto es muy desaconsejable...
